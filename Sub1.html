Aplicación del sistema de costeo por actividades (ABC) en una empresa manufacturera.

Objetivo de la subCompetencia

Demostrar la importancia de la contabilidad administrativa y los sistemas de información administrativos en el entorno actual, mediante la comunicación a través de la comunicación y compresión de consecuencias con respeto, colaboración y honradez.

Contabilidad administrativa y entorno empresarial:

El ejercicio muestra cómo una empresa (fabricante de iluminarias) utiliza el costeo ABC para adaptarse a un entorno competitivo, necesitando conocer con precisión el costo real de cada producto y no solo los costos primos.

Comparación con la contabilidad financiera:

Mientras la contabilidad financiera registraría los costos indirectos totales, la contabilidad administrativa (con el ABC) los analiza, asigna y rastrea hasta cada producto mediante actividades (supervisión, control de calidad, etc.), con el fin de mejorar la gestión interna.

Sistemas de información contemporáneos

El ABC es precisamente uno de estos sistemas. El ejercicio ejemplifica su mecanismo: se identifican las actividades que generan costos, se calcula una "tasa de actividad" (como $40 por hora de supervisión) y se asigna el costo a cada producto según el consumo que este hace de la actividad.

Propósito del ejercicio

Demostrar paso a paso cómo se distribuyen los costos indirectos entre varios productos usando generadores de costo específicos, para obtener una visión más exacta de la rentabilidad de cada uno y apoyar decisiones de precios, producción y mejora de procesos.

Ejercicio práctico

La compañía “Luz del Lentro” produce 4 tipos de iluminarias (exterior, interior, jardín y neón). A continuación se presenta información con respecto a cada una de ellas:

Producción (unidades) 18,000.00 40,000.00 11,000.00 4,000.00

Su taza por actividad:

  • Supervisión: 24,800 / (250 + 150 + 120 + 100) = 40
  • Control de calidad: 30,000 / (50 + 50 + 70 + 70) = 125
  • Movimiento de materiales: 1,600 / (5 + 8 + 12 + 7) = 50
  • Cambios de molde: 2,190 / (1 + 1 + 2 + 2) = 365

Costo indirecto

Exterior:

  • Supervisión: 250 X 40 = 10,000.00
  • Control de calidad: 50 X 125 = 6,250.00
  • Movimiento de materiales: 5 X 50 = 250.00
  • Cambios de molde: 2,190 / (1 + 1 + 2 + 2) = 365

Interior:

  • Supervisión: 150 x 40 = 6,000
  • Control de calidad: 50 x 125 = 6,250
  • Movimiento de materiales: 8 x 50 = 400
  • Cambios de molde: 1 x 365 = 365

Jardin:

  • Supervisión: 120 x 40 = 4,800
  • Control de calidad: 70 x 125 = 8,750
  • Movimiento de materiales: 12 x 50 = 600
  • Cambios de molde: 2 x 365 = 730

Neon:

  • Supervisión: 100 x 40 = 4,000
  • Control de calidad: 70 x 125 = 8,750
  • Movimiento de materiales: 7 x 50 = 365
  • Cambios de molde: 2 x 365 = 730
